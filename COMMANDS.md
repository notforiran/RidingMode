# Riding Mode Commands (V11 draft)

## Ride control
- `ride off`
- `motor off`
- `engine off`
- `stop riding mode`

## Music
- `play music`
- `play song`
- `pause music`
- `next song`
- `next track`
- `pre song`
- `pre music`
- `pre track`
- `volume up`
- `volume down`
- `volume max`
- `increase to max volume`

## Calls
- `call [name]`
- `first one`
- `second one`
- `find more`
- `cancel`
- `sim 1`
- `sim one`
- `simcard one`
- `first sim`
- `first simcard`
- `first`
- `sim 2`
- `sim two`
- `simcard two`
- `second sim`
- `second simcard`
- `second`
- `answer`
- `accept`
- `finish call`

## Notification mode
- `notif off`
- `notification off`
- `notif on`
- `notification on`

## Emergency mute
- `mute on`
- `mute off`

## Notes
- During an active call, only these three commands remain enabled: `finish call`, `end call`, `hang up`.
- When notification mode is off, Riding Mode only announces call-related events.
- Priority contacts added from the Contacts panel are checked before the main phonebook.

## V24 notes
- Listening is designed to be silent: no command confirmations for media/volume commands.
- Unknown recognition text is ignored in noisy motorcycle/traffic environments.
- Ride off variants include right of, rider of, write of, and bike off.
